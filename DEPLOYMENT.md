# Deployment Guide

See the complete deployment guide for Hostinger with Coolify in the project documentation.

## Quick Start

1. **Push to Git:** Ensure your code is in a Git repository
2. **Create MySQL Database** in Coolify
3. **Deploy Backend:** Point to `/server` directory
4. **Deploy Frontend:** Point to root directory
5. **Set Environment Variables** for both applications
6. **Initialize Database:** Run `node init-db.js` in backend container

For detailed instructions, refer to the full deployment guide.
